# Start/Sit Weekly Lineup Challenge

A simple fantasy-football-inspired Start/Sit decision game built with HTML, CSS, and JavaScript.

## Live Demo
(put your Netlify URL here once deployed)

## Features
- 4 matchups per slate (8 players total)
- Start/Sit picks per matchup
- Hint toggle (shows projected edge)
- Score Week: your score vs optimal score
- Win streak + best streak + best score (localStorage)
- Daily mode (same matchups for everyone each day, based on UTC date)
